Generic Data Format

In general, the AX.25 Information field can contain some or all of the following information:

  • APRS Data Type Identifier
  • APRS Data
  • APRS Data Extension
  • Comment

Generic APRS Information Field
  Data Type ID APRS Data APRS Data Extension Comment
Bytes: 1 n 7 n
